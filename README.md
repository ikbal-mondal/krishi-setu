# Krishi-Setu - Agricultural Trading Platform

**Live Link:** [Krishi-Setu Platform](https://krishi-setu-e2152.web.app)

## Overview

Krishi-Setu connects farmers and buyers through a modern web platform. It enables secure crop trading with features like interest management and real-time updates.

## Tech Stack

### Frontend

- React + Vite
- Tailwind CSS
- Firebase Auth
- React Router
- Context API
- Axios

### Backend

- Node.js + Express
- MongoDB Native Driver
- Firebase Admin SDK
- Environment Variables
- CORS enabled
